Simulates the electrostatic interactions of charged particles using Coulomb's law and Newtonian mechanics. Depending on friction and initial velocity, the particles naturally self-organize into stable geometric crystals or complex orbital shells. A mathematical distance-softening technique is applied to prevent the force equations from exploding when particles get extremely close to the center. Fading motion trails are used to visualize the chaotic orbital paths and Rutherford scattering events. To compile and edit the code you need gcc and raylib, though the stable 64-particle orbital version is already compiled in game.exe.
